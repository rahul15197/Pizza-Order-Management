# Pizza Order Management CLI Project
Pizza order management CLI system developed using Java programming language and Oracle Database Connectivity for storing the data about customers and orders.
Feel free to make changes and make it better.
Cheers!

## Overview

Accepts orders of a customer by showing a menu of pizzas. Generates a bill based on the orders given.

## Installation

Install ODBC (Oracle DB) driver for Database connectivity with Java.

## Usage

Use any Java IDE to run the program. Eclipse Mars was used to create this project.
